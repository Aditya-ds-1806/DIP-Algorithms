Implementation of DIP Algorithms in JS

1. Reducing number of color levels in an image

Problem

Transform an image \( f:X\times Y \to \{a,\dots, b\} \) to \( g:X\times Y \to \{c,\dots, d\} \)

Algorithm

  1. \( min = Min\{f(x,y) | 0 \leq x \leq m-1, 0 \leq y \leq n-1 \}\)
  2. \( g(x,y) = f(x, y) - min \)
  3. \(max = Max\{g(x,y) | 0 \leq x \leq m-1, 0 \leq y \leq n-1 \}\)
  4. \( g(x,y) = \dfrac{g(x, y)}{max} \)
  5. \( g(x,y) = (d-c)\cdot g(x,y) \)
  6. \( g(x,y) = g(x,y) + c \)
